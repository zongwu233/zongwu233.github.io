REST 以及 RESTful 辨析

从REST概念提出到现在已经20年了，业界的主流实践采纳借鉴了其部分原则，但与其最核心的架构设计渐行渐远。

实际上那个去中心化的、面向全世界开放的万维网时代正在落幕，构建在其基础之上的REST架构思想，也无可避免地被冷落或者被误解。
